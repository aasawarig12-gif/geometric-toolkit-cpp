# Geometric Toolkit (C++)

This is a menu-driven console application developed using Object-Oriented Programming (OOP) concepts in C++.

The program allows users to calculate the area and perimeter of various geometric shapes.

---

## Features

- Supports multiple geometric shapes:
  - Circle
  - Square
  - Rectangle
  - Equilateral Triangle
  - Scalene Triangle
  - Kite
  - Rhombus
  - Trapezium
- Menu-driven user interface
- Calculates both area and perimeter
- Uses dynamic object creation

---

## Concepts Used

- Inheritance
- Polymorphism (Virtual Functions)
- Abstraction using Base Class
- Dynamic Binding
- Classes and Objects

---

## Technologies Used

- C++
- Standard Libraries: iostream, cmath

---

## How to Run

1. Open the project in any C++ compiler (CodeBlocks / VS Code / Turbo C++)
2. Compile the program
3. Run the executable
4. Select a shape from the menu
5. Enter required values to get area and perimeter

---

## Example

User selects Circle → enters radius → program displays:
- Area of Circle
- Perimeter of Circle
